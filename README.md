# SpamDetector
An AI-powered system to detect and classify spam emails using machine learning.

## Features
- Detects Spam and Not Spam messages
- Simple GUI interface
- Built using Naive Bayes algorithm
- Trained on SMS Spam Collection Dataset

## Technologies Used
- Python
- Tkinter
- Pandas
- Scikit-learn

 ## How to Run
1. Install Python
2. Install required libraries

pip install -r requirements.txt

3. Run SpamDetector.py to train the model
4. Run spamGUI.py to launch the GUI
