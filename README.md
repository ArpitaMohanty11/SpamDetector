# SpamDetector
An AI-powered system to detect and classify spam emails using machine learning.

## Features
- Detects Spam and Not Spam messages
- Simple GUI interface
- Built using Naive Bayes algorithm
- Trained on SMS Spam Collection Dataset

## Technologies Used
- Python
- Tkinter
- Pandas
- Scikit-learn

## Sample Results

Spam:
"You won a free prize"

Not Spam:
"Let's meet tomorrow at 5 PM"
